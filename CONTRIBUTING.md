# Ay-Otonom-Navigasyon'a Katkıda Bulunma

Ay otonom navigasyon yığınına katkılarınızı bekliyoruz!

## Nasıl Katkıda Bulunulur?
1. Depoyu **Fork** edin.
2. Bir **Özellik Dalı** (Feature Branch) oluşturun (`git checkout -b feature/HarikaOzellik`).
3. Değişikliklerinizi **Commit** edin (`git commit -m 'HarikaOzellik eklendi'`).
4. Dalınızı **Push** edin (`git push origin feature/HarikaOzellik`).
5. Bir **Pull Request** açın.

## Teknik Standartlar
- Tüm kodlar **PEP 8** standartlarına uygun olmalıdır.
- Yeni özellikler `ay_otonom_navigasyon/tests/` altında birim testlerini içermelidir.
- Matematiksel fonksiyonlar LaTeX notasyonu ile dökümante edilmelidir.
